Chap 01_ 파이썬 들여다보기

프로그래밍 언어의 개념과 종류

  • 전산의 기본 개념: 소프트웨어⊃ 프로그램⊃ 어플리케이션 = 앱
  • 소프트웨어(Software): 프로그램 개발 과정에서 산출된 모든 것이 포함된 것. 프로그램과 프로그램이 정상적으로 실행될 수 있는 모든 부가적인 것을 포함!
    ≒ 라면 봉지: 라면에 대한 설명이 적힌 포장지이자, 내용물을 보호/보관하는 역할
  • 프로그램(Program): 컴퓨터에서 독립적으로 실행 가능한 모든 것
  • 어플리케이션(Application): 특정 목적을 위해 사용되는 프로그램
  • 앱(App): 어플리케이션의 줄임말
    예) 인터넷 앱 – 목적: 인터넷 / 문서 앱 – 목적: 문서 작성, 보기
  • 프로그래밍 언어가 반드시 가져야 하는 필수 구성요소: 문법, 단어
  • 정해진 ‘문법’ 규칙에 맞추어 약속된 ‘단어’를 사용하여 표현(구현)
  • 프로그래밍 언어의 단어: 키워드, 식별자, 연산자
  • 키워드(Keyword): 예약어(Reserved Word)라고도 하며, 문법적 의미를 미리 약속해둔 단어를 의미. 언어 차원에서 정해진 것.
  • 식별자(Identifier): 개발자가 직접 정의한 프로그래밍 구성요소의 이름. 변수, 함수, 클래스, 매소드, 패키지 등이 포함.
  • 연산자(Operator): R에서 설명한 연산자와 동일.
  • 프로그래밍 언어의 종류: C/C++, 자바(Java), 파이썬
  • 책에서는 HTML도 프로그래밍 언어라고 소개하고 있지만, HTML은 단순히 웹에서 사용되는 구조적 문서작성 기법일 뿐임.
  • 프로그래밍 언어별 특정 사용 목적
  • C : 하드웨어 제어/ 소프트웨어 레이어의 가장 밑단에서 성능을 조정 ⇒ 속도가 제일 빠름
  • C++: 윈도우에서 작동하는 거의 모든 프로그램을 만들 때 사용 예) 엑셀, 아래한글
  • 자바(Java): 앱 구현/서버 프로그램 만들 때
  • 파이썬(Python): 프로그래밍을 처음 배우는 사람에게 교육용으로 사용/ 데이터 분석, 인공지능 분야에 특화된 언어!

파이썬 소개

  • 파이썬 역사: 각 프로그래밍 언어의 역사를 알고 공부하세요,, - 이성구 강사님
  • 파이썬의 다양한 분류: 우리가 말하는 파이썬은 C Python으로 이해
  • C Python의 장점: C의 기능들을 파이썬으로 실행 가능하여, 속도가 빠름.
  • 파이썬의 특징 ★은 책에 나오지 않거나 강사님께서 강조하신 부분
  • 무료로 사용 가능하다
  • 직관적인 코드를 사용하여 읽기 쉽다:
  • 사물인터넷과 잘 연동된다: 라즈베리파이(사물인터넷 교육용 kit) 기반의 사물인터넷이 파이썬을 잘 지원하기 때문
  • 데이터 분석, 인공지능 분야에 특화된 외부 라이브러리가 풍부하다.
  • 웹 프레임워크를 사용해 강력하고 빠른 웹 환경 구축이 가능하다.
    ★ 파이썬은 객체 지향 언어다.
  • 자바, C++의 객체 지향 언어와는 차이가 있음
  • 파이썬은 모든 것이 다 객체이다. 객체를 무시하고 사용법대로만 배우면 나중에 못 써먹는 결과물이 나온다.
    ★ 파이썬은 동적 타입 언어다.
  • 동적 타입 언어란? 하나의 변수가 서로 다른 종류의 값을 저장하는 것
    예) R과 파이썬으로는 num=10, num=”ABC” 와 같이 하나의 변수에 다른 종류의 값 저장이 가능.
    But, C언어는 정적 타입 언어로 int num; (정수 값만 저장) 와 같이 변수 하나에 하나의 값 저장!
  • 변수가 상황/시점에 따라 달라질 수 있어 늘 변수의 종류를 검사 및 확인하면서 코딩을 진행해야 함.
    ★ 파이썬은 가독성이 뛰어난 언어다.
  • 문법적으로 들여쓰기가 정해져 있어 가독성이 뛰어남.
  • 주문장/부문장(Sub-statement)이 구분되어 사이에 들여쓰기가 있고, 이것을 지키지 않으면 코드가 실행되지 않음.
