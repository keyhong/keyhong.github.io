AWS Elastic Beanstalk

AWS

AWS Elastic Beanstalk

웹 애플리케이션 및 서비스를 배포하고 운영하는 서비스

구성 옵션 설정

  • 인프라 구성, 플랫폼 버전 변경 가능
  • 생성 시 EC2 인스턴스가 생성되고, 보안그룹, 탄력적 주소(elastic IP), auto scaling 그룹도 생성

Elastic-Beanstalk vs EC2

  • EC2와 달리, 사용자가 인프라 리소스를 구성하지 않고 애플리케이션 코드만 집중
  • 코드만 업로드하면 AWS가 용량 프로비저닝, 로드 밸런싱, Auto Scaling, 애플리케이션 상태 모니터링 등 배포를 자동 처리

지원 플랫폼

Java, .Net, PHP, Node.js, Python, Ruby, Go, Docker 웹 애플리케이션 지원

Elastic Beanstalk 사용하기

  1. 시작하기
  1. 환경구성
  1. 서비스 액세스 구성
  1. 네트워킹, 데이터베이스 및 태그 설정
  1. 인스턴스 트래픽 및 조정 구성
  1. 업데이트, 모니터링 및 로깅 구성
  1. 검토
  1. 생성 확인
