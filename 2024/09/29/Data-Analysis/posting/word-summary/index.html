광고 용어

  • CPC (Cost Per Click) : 온라인 광고 가격을 책정하는 대표적인 방법. 사용자가 검색 결과로 보여진 광고를 클릭하여 광고주의 사이트로 이동했을 때에 광고비가 소진되는 과금 체계

  • CPM (Cost Per Mile) : 온라인 광고 가격을 책정하는 방법 중 하나로, 1000회 광고 노출을 기준으로 가격을 책정하는 과금 체계

  • Impression : 노출. 광고가 사용자에게 보여지는 것 또는 검색 사용자에게 광고가 노출된 횟수

  • CTR (Click Through Rate) : 해당 광고가 특정 기간동안 클릭된 횟수를 노출된 횟수로 나눈값을 백분율로 표시한 값

  • 페이지 RPM : 총 수입을 페이지 조회수 1000회로 나눈 값. 1000회 노출당 페이지 수익(RPM)은 예상 수입을 페이지 조회수로 나눈 다음 1000을 곱한 값. (예상수입/페이지 조회수) * 1000

  • 평균클릭비용. CPC=Cost Per Click, PPC=Pay Per Click. 전체 광고비를 클릭 수로 나눈 값으로 클릭 1회당 단가. (광고비/클릭) 광고 효과를 측정하는 지표 중 하나

  • 랜딩페이지 : 인터넷 사용자가 광고를 클릭하고 광고주의 사이트에 방문했을 때 최초로 보게 되는 웹 페이지

  • 페이지뷰 : 홈페이지에 들어온 접속자가 둘러 본 페이지

  • 문맥광고 : 페이지 내 컨텐츠를 분석하여 관련 키워드를 추출하고 광고를 노출하는 방식

유저 관련 지표

  • UV (Uniuqe View) : 유저의 고유 카운트

  • AU (Active User) : 접속한 유저

  • RU (Registerd User) : 신규유저 / 게임 설치 뿐만 아니라 계정 등록을 한 유저

  • NPU (New Paid User) : 신규 구메(과금) 유저

  • NNPU : 신규 가입하여 신규 구매한 유저

  • RPU : 기존 가입 유저 중 구매 이력이 없던 유저의 구매 전환 수

과금 관련 지표

  • PUR (Paid User Rate) : 접속 유저 대비 결제 유저 비율

  • NPUR (New Paid User Rate) : 접속 유저 대비 신규 결제 유저 비율

  • Revenue : 실제 발생한 매출

  • ARPPU (Average Revenue Per Paid User) : PU 1명 당 결제액

    • 매출(Sale) / PU 수

  • ARPAU (Average Revenue Per Active User) : AU 1명 당 결제액

    • 매출(Sales) / AU 수

모객 관련 지표

  • RET (Retension) : 잔존율, 유저 잔존 여부

  • LTV (Life Time Value) : 유저 1명 당 생애 누적

    • 누적 매출 (Sales) / RU

    • 일평균 지불금액(ARPDAU) x 지속방문일 (LT, Life Time)

  • CPI (Cost Per Install) : 게임 Install 한번 당 지출 비용

    • Cost / Install 수

  • CPRU (Cost Per RU) : RU 당 지출 비용

    • Cost / RU 수

기타

  • IAP (In App Purchase) : 신용카드를 통한 인 앱 결제

  • IAA (In App Ads) : 인 앱 광고로 발생되는 매출

광고 관련 지표

  • Impresssion : 광고가 노출된 횟수

  • CPM (Cost Per Mile) : 1000개 노출 당 광고 비용

유저 및 게임

  • UV (Unique View) : 순 방문자

  • MUV (Multigame Unique Visitor) : 특정 기간 클라이언트 혹은 애플리케이션을 내려받아 로그인 및 플레이한 방문자

  • AU (Active User) : 활성 사용자를 가리키며 일반적으로 DAU와 동일시

  • MAU (Monlth Active User) : 한달 내 게임 순 방문자 수

  • RU (Registerd User) : 전체 가입자 수

  • NRU (New Registered User) : 기간 내 순 신규 가입자 수

    • user_id : int : 각 사용자의 ID (기본 키)
    • username : varchar(50) : 사용자의 이름
    • email : varchar(100) : 사용자의 이메일 주소 (고유)
    • registration_date : datetime : 사용자가 등록한 날짜
    • last_login_date : datetime : 사용자가 마지막으로 로그인한 날짜
SELECT DATE_FORMAT(registration_date, '%Y-%m') AS registration_month
     , COUNT(user_id) AS nru
FROM users
WHERE registration_date BETWEEN '2024-01-01' AND '2024-01-31'
GROUP BY DATE_FORMAT(registration_date, '%Y-%m')
ORDER BY registration_month
;

  • DL(Download) : 기간 내 다운로드 횟수

  • TS(Time Spent) : 게임에 방문한 이용자의 평균 체류 시간. ((시그마)로그아웃 시간 - 로그인 시간) / MU

  • CCU (Concurrent User) : 특정 시간의 동시 접속자 수

    • session_id : int : 세션ID
    • user_id : int : 사용자ID
    • session_start : datetime : 세션시작시간
    • session_end : datetime : 세션종료시간(선택)
SELECT COUNT(DISTINCT user_id) AS ccu
FROM user_sessions
WHERE session_start <= '2024-09-29 15:00:00' -- 측정할 시점
  AND (session_end IS NULL OR session_end >= '2024-09-29 15:00:00');
  • MCU (Max Concurrent User) 혹은 PCU (Peak Concurrent User) : 기준 단위 동시 접속자 최대값 (예: 5분)
SELECT MAX(ccu) AS PCU
FROM (
    SELECT COUNT(DISTINCT user_id) AS ccu
    FROM user_sessions
    WHERE start_session <= '2024-09-29 23:59:59' -- 하루의 끝
    AND (session_end IS NULL OR session_end >= '2024-09-29 00:00:00'); -- 하루의 시작
    GROUP BY DATEFORMAT(start_session, '%Y-%m-%d %H:00:00') -- 시간 단위 그룹화
    -- GROUP BY DATEFORMAT(start_session, '%Y-%m-%d %H:%i:00') -- 분 단위 그룹화
)
  • ACU (Average Concurrent User) : 기준 단위 동시 접속자 평균값 (예: 5분)
SELECT AVG(ccu) AS PCU
FROM (
    SELECT COUNT(DISTINCT user_id) AS ccu
    FROM user_sessions
    WHERE start_session <= '2024-09-29 23:59:59' -- 하루의 끝
    AND (session_end IS NULL OR session_end >= '2024-09-29 00:00:00'); -- 하루의 시작
    GROUP BY DATEFORMAT(start_session, '%Y-%m-%d %H:00:00') -- 시간 단위 그룹화
    -- GROUP BY DATEFORMAT(start_session, '%Y-%m-%d %H:%i:00') -- 분 단위 그룹화
)
  • DAU (Daily Active User) : 하루 내 게임 순 방문자 수
SELECT event_date
     , COUNT(DISTINCT user_id) AS DAU
FROM fact_user_session_day
WHERE event_date BETWEEN DATE_SUB(NOW(), INTERVAL 1 MONTH) AND NOW()
GROUP BY event_date
ORDER BY event_date DESC
;
  • WAU (Weekly Active User) : 한 주 내 게임 순 방문자 수
SELECT fo.event_date
     , COUNT(DISTINCT ft.customer_id) AS WAU
FROM fact_user_session_day AS fo
    LEFT JOIN fact_user_session_day AS ft
           ON ft.event_date BETWEEN ft.event_date AND DATE_SUB(ft.event_date, INTERVAL 6 DAY)
GROUP BY fo.event_date
  • Stickiness
SELECT fo.event_date
     , COUNT(DISTINCT fo.customer_id) AS DAU
     , COUNT(DISTINCT ft.customer_id) AS WAU
     , COUNT(DISTINCT fo.customer_id) / COUNT(DISTINCT ft.customer_id) AS stickiness
FROM fact_user_session_day AS fo
    LEFT JOIN fact_user_session_day AS ft
           ON ft.event_date BETWEEN ft.event_date AND DATE_SUB(ft.event_date, INTERVAL 6 DAY)
GROUP BY fo.event_date
  • ARPDAU (Average Revenue Per Daily Active User)
SELECT event_date
     , ROUND(SUM(revenue) / COUNT(DISTINCT user_id), 2) AS ARPDAU
FROM fact_user_session_day
WHERE event_date BETWEEN DATE_SUB(NOW(), INTERVAL 1 MONTH) AND NOW()
GROUP BY event_date
ORDER BY event_date DESC
;
