Installation

Publish Date:   2024-02-01
Flink

Env

  • H/W : 1 Virtual Machine
  • OS : Amazon Linux release 2 (Karoo)

아카이브 파일 수동 설치

설치순서

  1. 패키지 색인 파일(패키지 저장소가 제공하는 패키지에 대한 정보 파일) 업데이트
# Debian 계열 리눅스(Ubuntu)
$ sudo apt-get update

# RedHat 계열 리눅스(CentOS, RHEL)
$ sudo yum update
  1. 자바(openjdk 설치) 및 환경변수 등록
# Debian 계열 리눅스(Ubuntu)
$ sudo apt-get install openjdk-11-jdk

# RedHat 계열 리눅스(CentOS, RHEL)
$ sudo yum install -y java-11-openjdk-devel.x86_64

# .bashrc에 환경변수 등록
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
export PATH=$PATH:$JAVA_HOME/bin
  1. Flink 설치

Apache Flink Installation

# 플링크 다운로드
$ wget https://dlcdn.apache.org/flink/flink-<x.y.z>/flink-<x.y.z>-bin-scala_<a.b>.tgz

# 다운받은 압축파일 풀기
$ tar -zxvf flink-<x.y.z>-bin-scala_<a.b>.tgz -C <경로>
  1. 로컬 클러스터 시작 및 중지
# 로컬 클러스터 실행
$ .bin/start-cluster.sh

# daemon process 확인
$ ps aux | grep flink
  1. 작업 제출하기
# 작업 제출
$ ./bin/flink run examples/streaming/WordCount.jar

# 작업 결과 보기
$ tail log/flink-*-taskexecutor-*.out

도커 이미지로 설치

# flink 이미지 다운로드
$ docker pull flink

# job manager 
$ docker run -d --name jobmanager -e JOB_MANAGER_RPC_ADDRESS=jobmanager -p 8081:8081 flink jobmanager

# task manager1, 2 실행
$ docker run -d --name taskmanager-1 --link jobmanager:jobmanager -e JOB_MANAGER_RPC_ADDRESS=jobmanager flink taskmanager
$ docker run -d --name taskmanager-2 --link jobmanager:jobmanager -e JOB_MANAGER_RPC_ADDRESS=jobmanager flink taskmanager

Flink job 실행하기

Flink 클러스터를 설치했으면 job을 제출해보려 한다. Flink 아카이브에는 테스트를 할 수 있는 jar파일이 example 폴더에 있다. Flink job의 실행은 bin 폴더 안의 flink 파일을 통해 run 커맨드를 실행한다. WordCount.jar 파일을 실행시키는 다음 예시를 보자.

# 플링크 아카이브로 접근하기 위해 도커 컨테이너 내부 진입
$ docker exec -it jobmanager /bin/bash

# example 폴더에 있는 WordCount.jar 파일을 job으로 제출
$ ./bin/flink run examples/streaming/WordCount.jar

# bin/flink의 기본 사용 Syntax
Syntax: run [OPTIONS] <jar-file> <arguments>
