How to connect to adapters

어댑터는 dbt의 필수 구성 요소입니다. 가장 기본적인 수준에서는 dbt가 다양한 지원 데이터 플랫폼에 연결하는 방법입니다. 보다 높은 수준에서 어댑터는 분석 엔지니어에게 보다 이동 가능한 기술을 제공하고 분석 프로젝트의 구조화 방식을 표준화하기 위해 노력합니다. 새로운 직장으로 이동할 때 다른 데이터 플랫폼을 사용하는 경우 새로운 언어나 SQL 변형을 배워야 했던 시절은 지나갔습니다. 이것이 dbt의 어댑터의 힘입니다 - 자세한 내용은 Build, test, document, and promote adapters 가이드를 참조하세요.

이 섹션에서는 dbt를 어댑터에 연결하는 다양한 방법과 유지 관리자가 무엇인지에 대해 더 자세히 설명합니다.

Set up in dbt Cloud

dbt를 배포하는 가장 빠르고 신뢰할 수 있는 방법을 알아보세요. dbt Cloud는 조직 전체에 걸쳐 dbt Core를 실행하는 호스팅 아키텍처로, dbt Cloud UI에서 직접 다양한 [trusted] 데이터 플랫폼 제공업체와 원활하게 connect 할 수 있습니다.

Install with dbt Core

명령줄을 사용하여 로컬에 오픈 소스 도구인 dbt Core를 설치하세요. dbt는 각각 전용 어댑터 플러그인을 사용하여 여러 다른 데이터 플랫폼과 통신합니다. dbt Core를 설치할 때 특정 데이터베이스에 대한 어댑터도 설치해야 하며, connect to dbt Core,하고 profiles.yml 파일을 설정해야 합니다.

일부 예외를 제외하고는(1), 모든 adapters를 PyPI에서 python -m pip install adapter-name을 사용하여 설치할 수 있습니다. 예를 들어, Snowflake를 설치하려면 python -m pip install dbt-snowflake 명령을 사용하세요. 설치에는 dbt-core와 필요한 모든 종속 항목이 포함되며, 다른 종속 항목뿐만 아니라 다른 어댑터 플러그인도 포함될 수 있습니다. installing dbt에 대해 자세히 알아보세요.

(1) pip를 사용하여 설치할 때 PyPI 패키지 이름을 사용하세요.

Adapter repo name PyPI package name
dbt-layer dbt-layer-bigquery
