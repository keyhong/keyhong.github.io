dbt Quickstarts

dbt 여정을 시작하려면, dbt Cloud 또는 dbt Core를 variety of data platforms과 함께 설정하는 데 도움을 주는 단계별 가이드가 포함된 퀵스타트 중 하나를 시도해 보세요.

dbt Cloud

dbt Cloud는 단일의 완전 관리 소프트웨어 서비스를 사용하여 데이터 제품을 개발, 테스트, 배포 및 탐색할 수 있는 확장 가능한 솔루션입니다. 이 솔루션은 다양한 기술을 가진 팀이 모든 규모에서 신뢰할 수 있는 데이터 제품을 구축할 수 있도록 하며, 다음과 같은 기능을 포함합니다:

  • 다양한 사용자에게 맞춘 개발 환경 (브라우저 내 dbt Cloud IDE 또는 로컬에서 dbt Cloud CLI 사용)

  • 기본 제공 CI/CD workflows

  • 모든 엔드포인트에 제공될 수 있는 일관된 메트릭을 위한 dbt Semantic Layer

  • 멀티 프로젝트 dbt Mesh 설정을 통한 데이터의 도메인 소유권

  • 협업 데이터 발견과 이해를 위한 dbt Explorer


dbt Cloud features에 대해 자세히 알아보고 오늘 무료 체험을 시작하세요.


Quickstart for dbt Cloud and Azure Synapse Analytics (Preview)

  • dbt Cloud를 Azure Synapse Analytics와 통합하여 데이터 변환 작업을 수행하는 방법을 알아보세요.

Quickstart for dbt Cloud and BigQuery

  • dbt Cloud를 BigQuery와 결합하여 분석 워크플로우를 간소화하는 방법을 알아보세요.

Quickstart for dbt Cloud and Databricks

  • dbt Cloud를 Databricks와 통합하여 효율적인 데이터 처리 및 분석을 어떻게 할지 배워보세요.

Quickstart for dbt Cloud and Microsoft Fabric

  • dbt Cloud와 Microsoft Fabric 간의 시너지를 탐색하여 데이터 변환을 최적화하세요.

Quickstart for dbt Cloud and Redshift

  • 더 유연한 데이터 변환을 위해 dbt Cloud를 Redshift에 연결하는 방법을 배워보세요.

Quickstart for dbt Cloud and Snowflake

  • 데이터 변환에 dbt Cloud를 Snowflake와 함께 사용하여 전체 잠재력을 발휘해보세요.

Quickstart for dbt Cloud and Starburst Galaxy

  • Starburst Galaxy와 함께 dbt Cloud를 활용하여 데이터 변환 워크플로우를 향상시켜보세요.

dbt Core

dbt Core는 데이터 전문가들이 분석 엔지니어링 최고의 실천 방법을 사용하여 데이터를 변환할 수 있도록 하는 명령줄 open-source tool입니다. 이는 수동 설정과 사용자 정의를 선호하는 개인 및 소규모 기술 팀에 적합하며, 커뮤니티 어댑터 및 오픈 소스 표준을 지원합니다.

다음 퀵스타트를 참조하여 dbt Core를 시작하세요:

  • dbt Core를 설치하고 프로젝트를 설정하는 방법을 배우는 dbt Core from a manual install입니다.

  • 코드스페이스를 생성하고 dbt build 명령을 실행하는 방법을 배우는 dbt Core using GitHub Codespace입니다.

Related docs

이 추가 자원들을 활용하여 dbt 지식과 전문성을 확장하세요:

  • Join the bi-weekly demos는 dbt Cloud를 실제로 보고 질문할 수 있는 데모에 참여하는 것입니다.

  • dbt Cloud AWS marketplace에는 dbt Cloud를 AWS에 배포하는 방법, 사용자 리뷰 등의 정보가 포함되어 있습니다.

  • Best practices에는 현재 구조, 스타일 및 설정에 대한 dbt Labs의 접근 방식에 대한 정보가 포함되어 있습니다.

  • dbt Learn은 dbt 기본 개념, 고급 주제 등을 다루는 무료 온라인 강좌를 제공합니다.

  • Join the dbt Community는 전 세계의 다른 데이터 전문가들이 dbt를 어떻게 활용하고 있는지 배우고, 여러분의 경험을 공유하며, dbt 프로젝트에 도움을 받는 방법을 배우는 것입니다.
