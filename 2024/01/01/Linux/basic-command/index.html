[Linux] Basic Command

Publish Date:   2024-01-01
Linux

파일/폴더 조회

# ls : list
$ ls [-option]
    -l  # line          : 해당 경로의 모든 파일을 라인으로 자세하게 출력
    -a  # all           : 해당 경로의 모든 파일 출력 (숨김 파일 보기)
    -R  # recursive     : 하위 디렉토리의 내용까지 출력
    -F  #               : 파일 특성에 따라 끝에 특수문자 붙여 출력

# 디렉토리 내 파일들의 정보를 화면 단위로 출력
$ ls -al | more

파일/폴더 삭제

# rm : remove
$ rm [-option]
    -r, -R   # recursive          : 하위 디렉토리를 포함한 모든 파일 삭제
    -f, --f  # force              : 삭제 여부를 묻지않고 강제 삭제

파일 종류 확인하기

$ file <파일명>

압축 / 압축해제

# tar : tape archive
$ tar [-option]
    -c, --c  # create                : 파일을 tar로 압축
    -C       # (Change directory)    : 압축 또는 압축해제하여 저장할 파일이 있는 경로 이동하여 실행
    -x       # extract               : tar 압축풀기
    -f       # file                  : 파일 이름을 지정
    -z       #                       : gzip으로 압축하거나 gzip(tar.gz) 해제

# tar 압축을 할 때
$ tar -cf [압축파일명.tar] [폴더명]

# tar 압축을 풀 때
$ tar -xf [압축파일명.tar]

# tar.gz 압축을 할 때
$ tar -zcf [압축파일명.tar] [폴더명]

# tar.gz 압축을 풀 때
$ tar -zxf [압축파일]

시스템 종료

$ shutdown [-option] [종료시킬 시간]
  -c       # cancel                : 예약된 shutdown 명령 취소
  h       # halt                  : 시스템을 멈추고 종료
  r       # restart               : 시스템 종료 후 재시작

$ halt

$ init 0

시스템 재시작

$ shutdown -r [종료시킬 시간]

# 파일 시스템 unmout, 시스템 shutdown, 시스템 실행수준(run-level) 3으로 변경
$ reboot

조건을 만족하는 문자열 찾기

$ grep [-option] <문자열>
    -v       # 문자열이 포함되지 않은 행 찾기
    -f       # 하위디렉토리까지 모두 포함하여 문자열을 포함한 파일 찾기

두 파일 비교하기

# compare
$ cmp <file1> <file2>

# difference
$ diff <file1> <file2>
    - i        # 대소문자 구분하기

현재 접속되어 있는 모든 사용자에게 메세지 보내기 (Broadcast message)

$ wall

[메세지 입력]

ctrl + D

이동식 보조 기억장치(CD-ROM, DVD 등)의 미디어를 꺼낼 때 사용

$ eject

ftp에서 여러 개의 파일을 다운로드 받기

$ mget

현재 시스템에 접속한 사용자 계정 조회

$ users

파일 시스템 검사 및 오류 수정

$ fsck

용량이 큰 파일을 여러 개의 작은 파일로 분리

$ split

자신의 호스트명을 변경 또는 확인

$ host

문자열 치환

$ sed -i "%s/[변경대상 문자열]/[변경 후 문자열]/g" [변경할 파일]
