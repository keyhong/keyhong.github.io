Publish Date:   2024-01-01
Linux

리눅스 환경변수

시스템 또는 사용자 수준에서 사용되는 설정 정보를 저장하는데 사용되는 변수

시스템 구성

시스템 전반적인 설정을 저장

  • PATH 환경변수는 실행 파일의 경로를 지정하여 명령어를 실행할 때 프로그램을 찾을 수 있게함함

  • PATH 환경변수 추가하기 (.bashrc 또는 .bash_profile)

.bashrc
export PATH=$PATH:<추가할 경로>

  • /etc (시스템 전역 설정 파일을 포함하는 디렉토리) 내 중요 파일

    • group : 서버의 그룹 id 모음

    • hostname : 서버의 hostname

    • hosts : 서버의 hosts

    • passwd :서버의 passwd (신규 생성 유저 description)

사용자 환경

사용자 특정 설정을 저장

  • HOME 환경변수는 사용자의 홈 디렉토리 경로를 지정
디렉토리 구조
[ / ]
  |
  |--- [ user 1 (/home/user1) ]
  |         |
  |         |- .bashrc
  |         |
  |         |- .bashprofile
  |
  |--- [ user 2 (/home/user2) ]
  |
  |--- [ user OOO (Home directory folder) ]
  | 
  |--- [ etc (/etc) ]
            |
            |- .bashrc
            |
            |- .bash_profile

프로그램 구성

프로그램이 실행될 때 필요한 구성 정보를 저장

  • 프록시 설정, 로케일 설정 등이 포함

  • spark의 spark-env.sh 이나 airflow의 airflow.cfg 등의

시스템 시작시 환경변수 실행 순서

  1. /etc/profile
  2. /etc/profile.d/*.sh
  3. ~/.bash_profile
  4. ~/.bashrc
