Vector

벡터가 뭔가요?

벡터는 숫자를 원소로 가지는 리스트(list) 또는 배열(array)

# 열 벡터	
    1
x = 7
    2

# 행 벡터	
Xt = [1, 7, 2]

# 행 벡터 생성
x = [1, 7, 2]
x = np.array([1, 7, 2])

벡터에 있는 숫자의 개수는 벡터의 차원

x = [x1, x2, ..., xd]
  • 벡터는 공간에서 한 점을 나타냄 (1차원, 2차원, 3차원, …, n차원)
  • 벡터는 원점(제로벡터, 영벡터)으로부터 상대적 위치를 표현
  • 벡터에 숫자를 곱해주면 길이만 변함
    
    # a > 1 : 1보다 크면 길이가 늘어남
    # 0 < a < 1 : 1보다 작으면 길이가 줄어듬
    # a < 0 : 0보다 작으면 반대 방향이 됨

    # 스칼라곱
    ax = ax = [ax1, ax2, ax3, ..., axd]

- 벡터끼리 같은 모양을 가지면 덧셈, 뺼셈을 계산할 수 있음

```python
x = [x1, x2, x3, ..., xd]
y = [y1, y2, y3, ..., yd]
x +- y = [x1+-y1, x2+-y2, x3+-y3, ..., xd+-yd]
  • 벡터끼리 같은 모양을 가지면 성분곱(Hadamard product)을 계산할 수 있음

x = [x1, x2, x3, ..., xd]
y = [y1, y2, y3, ..., yd]
x*y = [x1*y1, x2*y2, x3*y3, ..., xd*yd]


>>> import numpy as np
>>> x = np.array([1, 7, 2])
>>> y = np.array([5, 2, 1])
>>> x + y
array([6, 9, 3])

>>> x -y 
array([-4, 5, 1])

>>> x * y
array([5, 14, 2])

  • 두 벡터의 덧셈은 다른 벡터로부터 상대적 위치이동을 표현

  • 두 벡터의 뺄셈은 마이너스 벡터를 더하는 것과 같이 표현 가능

  • 벡터의 노름 구해보기

  • 벡터의 노름(norm)은 원점에서부터의 거리를 말함

# || ~ || 기호는 노름(norm)이라 부름
# 노름은 임의의 차원 d에 대해서 성립

x = [x1, x2, x3, ..., xd]
  • L1-노름은 각 성분의 변화량의 절대값을 모두 더함
  • L1-노름 상의 : {x: ||x||1 = 1}
  • 예: Robust 학습, Lasso 회귀

# ||x||1 = 시그마 (1~d) |xi|

def l1_norm(x):
    x_norm = np.abs(x)
    x_norm = np.sum(x_norm)
    
    return x_norm
  • L2-노름은 피타고라스 정리를 이용해 유클리드 거리를 계산함
# L2-노름은 np.linalg.norm을 이용해도 구현 가능
  • L2-노름 상의 원 : {x: ||x||2 = 1}
  • 예: Laplace 근사, Ridge 회귀
# ||x||2 = 루트 (시그마 (1~d) |x1| ** 2)

def l2_norm(x):
    x_nrom = x * x
    x_norm = np.sum(x_norm) 
    x_norm = np.sqrt(x_norm)
    
    return x_norm

  • 왜 다른 노름을 소개하나요?

  • 노름의 종류에 따라 기하학적 성질이 달라짐

  • 머신러닝에선 각 성징들이 필요할 때가 있으므로 둘 다 사용

  • 두 벡터 사이의 거리르 구해보자!

  • L1, L-노름을 이용해 두 벡터 사이의 거리를 계산할 수 있음

  • 두 벡터 사이의 거리를 계산할 때는 벡터의 뺄셈을 이용

  • 뺄셈을 거꾸로 해도 거리는 같음

# ||y-x|| = ||x-y||
  • 두 벡터 사이의 각도 구해보기
  • 두 벡터 사이의 거리를 이용하여 각도도 계산할 수 있음
    ** 두 벡터 사이의 각도는 L1-노름이 아닌 L2-노름(||~||2)에서만 계산 가능
  • 제2 코사인 법칙에 의해 두 벡터 사이의 각도를 계산할 수 있음
  • 분자를 쉽게 계산하는 방법이 내적
# x와 y의 거리 = ||x-y||2 = ||y-x||2

"""
        ||x|2 ** 2 + ||y||2 ** 2 - ||x-y||2 ** 2 
cos세타 = -----------------------------------------
                2 * ||x||2 * ||y|||2


                    2 * <x, y>
    = -----------------------------------------
                2 * ||x||2 * ||y||2
                
                
                        <x, y>
    = -----------------------------------------
                    ||x||2 * ||y||2					
                    
                        <x, y>
    = -----------------------------------------
                    ||x||2 * ||y||2

"""

# <x, y> = 시그마 (1~d) xi * yi
                    
# ** 이 연산을 내적(inner product)라 부른다

def angle(x, y):
    v = np.inner(x, y) / (l2_norm(x) * l2_norm(y))
    theta = np.arccos(v)
    
    return theta
