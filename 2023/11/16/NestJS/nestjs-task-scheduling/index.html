[NestJS] Task Scheduling

NestJS

@nestjs/schedule

리눅스의 cron과 같은 작업 예약 패키지이다. @nestjs/schedule 패키지는 Nodejs.cron 패키지를 통합한 패키지이다.

패키지 설치

$ npm i --save @nestjs/schedule

모듈 임포트

import { ScheduleModule } from '@nestjs/schedule';

사용법

  1. app.module.ts에서 라이브러리를 임포트 한다.
import { ScheduleModule } from '@nestjs/schedule';
  1. imports의 array 안에 SchedulerModule.forRoot()를 넣는다. forRoot()가 호출되면 스케줄러 초기화와 declarative cronjob, timeouts, interval을 등록한다.
@Module({
    imports: [
        ScheduleModule.forRoot()
    ],
	...
})
1+2 전체 코드
import { Module } from '@nestjs/common';
import { AppService } from './app.service';
import { ScheduleModule } from '@nestjs/schedule';

@Module({
    imports: [
        ScheduleModule.forRoot()
    ],
    controllers: [],
    providers: [AppService],
})
export class AppModule {}

declarative cronjob

크론 작업을 통해 특정 메서들 어떤 시간에 한번 실행하거나 특정 주기마다 반복적으로 실행하게 만들 수 있다. 크론 작업으로 설정할 메서드는 @Cron() 데코레이터를 이용해 선언 할 수 있다.

import { Injectable, Logger } from '@nestjs/common';
import { Cron } from '@nestjs/schedule';

@Injectable()
export class TasksService {
  private readonly logger = new Logger(TasksService.name);
  
  // 크론 잡 선언
  @Cron('45 * * * * *')
  handleCron() {
    this.logger.debug('Called when the current second is 45');
  }
}

@Cron()의 소괄호 사이에는 작업을 실행할 특정 시간을 인자로 넣을 수 있다. 인자로 넣을 수 있는 패턴은 3가지 이다.

  1. 일반적으로 리눅스에서 제공하는 표준 크론 패턴을 기본적으로 제공하고 있다. 원하는 표현식을 만들어주는 크론 메이커와 같은 사이트가 있다.

  2. nestjs@schedule 패키지에는 일상적으로 사용되는 크론 패턴을 편리하게 사용 할 수 있도록 만든 열거형을 제공하고 있다. 미리 정의된 열거형은 크론 형식과 매핑되어 있다. 열거형으로 인자를 사용하는 코드는 아래와 같다.

import { Injectable, Logger } from '@nestjs/common';
import { Cron, CronExpression } from '@nestjs/schedule'; // 열거형 사용을 위해서 CronExpression 임포트

@Injectable()
export class TasksService {
  private readonly logger = new Logger(TasksService.name);
  
  // 크론 잡 선언 (열겨형 enum을 이용하여 매초마다 스케줄링 실행)
  @Cron(CronExpression.EVERY_SECOND)
  handleCron() {
    this.logger.debug('Called when the current second is 45');
  }
}
  1. 마지막으로 Date 객체를 넣는 방법이 있다.
// 앱이 시작하고 10초후 크론 잡 실행
@Cron(new Date(Date.now() + 10 * 1000))

@Cron 데코레이터에 추가 옵션을 두 번째 인자로 넣기 (key: value)

| name | 크론 잡의 이름 설정
| timeZone | 실제 시간을 타임존에 맞는 시간으로 변환. 사용가능한 타임존은 MOMENT TIMEZONE에서 확인 가능
| utcOffset | timeZone 대신 사용자가 직접 타임존 오프셋을 지정
| disabled | 작업을 실행시키지 말지를 결정

import { Injectable } from '@nestjs/common';
import { Cron, CronExpression } from '@nestjs/schedule';

@Injectable()
export class NotificationService {
  @Cron('* * 0 * * *', {
    name: 'notifications',
    timeZone: 'Europe/Paris',
  })
  triggerNotifications() {}
}
