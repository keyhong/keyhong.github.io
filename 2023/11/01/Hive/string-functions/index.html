String Functions

Publish Date:   2023-11-01
Hive

☆ CONCAT

두 문자열을 결합하여 반환한다.

-- Syntax: CONCAT (string|binary A, string|binary B...)
SELECT CONCAT(col1, '-', col3) FROM table1;

LENGTH

컬럼에 있는 개별 값들의 문자열의 길이를 반환한다.

-- Syntax: LENGTH (string A)
SELECT LENGTH(col3) FROM table1;

LOWER

모든 대문자를 소문자로 변환한 문자열을 반환한다.

-- Syntax: LOWER (string A)
SELECT LOWER(col3) FROM table1;

UPPER

모든 소문자를 대문자로 변환한 문자열을 반환한다.

-- Syntax: UPPER (string A)
SELECT UPPER(col3) FROM table1;

LPAD

원하는 문자를 문자열의 길이를 만족할 때까지 왼쪽부터 채워서 반환한다.

-- Syntax: LPAD (string str, int len, string pad)
SELECT LPAD(col3, 10, 'v') FROM table1;

-- Return --
vvvEngland
vvvEngland
vvvvvWales

RPAD

원하는 문자를 문자열의 길이를 만족할 때 까지 오른쪽부터 채워서 반환한다.

-- Syntax: RPAD (string str, int len, string pad)
SELECT RPAD(col3, 10, 'v') FROM table1;

-- Return --
Englandvvv
Englandvvv
Walesvvvvv

LTRIM

문자열 안의 공백을 왼쪽부터 모두 잘라내어 반환한다.

-- Syntax: LTRIM (string A)
SELECT LTRIM('       jivesh');

-- Return --
'jivesh'

RTRIM

문자열 안의 공백을 오른쪽부터 모두 잘라내어 반환한다.

-- Syntax: RTRIM (string A)
SELECT 	RTRIM('jivesh       ');

-- Return --
'jivesh'

REPEAT

원하는 문자열을 원하는 개수만큼 반복해서 결합한 문자열을 반환한다.

-- Syntax: REPEAT (string str, int n)
SELECT REPEAT(col3, 2) FROM table1;

-- Return --
EnglandEngland
EnglandEngland
WalesWales

REVERSE

문자열을 뒤집어서 반환한다.

-- Syntax: REVERSE (string A)
SELECT REVERSE(col3) FROM table1;

-- Return --
dnalgnE
dnalgnE
selaW

☆ SPLIT

구분 기호나 특수 문자를 기준으로 문자열을 분할한다.

-- Syntax: SPLIT (string str, string pat)
SELECT SPLIT('hive:hadoop', ':');

-- Return -- 
["hive", "hadoop"]

☆ SUBSTR

문자열에서 주어진 인덱스만큼의 문자열을 빼내어 반환한다. 하이브에서 첫번째 문자의 인덱스는 1부터 시작한다.

-- Syntax: SUBSTR (string|binary A, int start)
SELECT SUBSTR('hive is querying tool', 4);

-- Return --
'e is querying tool'

SELECT SUBSTR('hive is querying tool', 4, 3);

-- Return --
'e i'

☆ INSTR

문자열에서 주어진 특정 문자나 문자열의 위치를 인덱스로 반환한다.

-- Syntax: INSTR (string str, string substr)
SELECT SUBSTR('hive is querying tool', 'e');

-- Return --
4
