Hive

Hive 설치하기

  • 하이브는 사용자의 워크스테이션에서 실행, 작성된 SQL 쿼리는 일련의 맵리듀스 잡으로 변환되어 하둡클러스터에서 구동

  • HDFS에 저장된 데이터(directory/file)에 구조(Schema)를 입히는 방식으로 데이터를 테이블로 구조화

  • 테이블 스키와 같은 메타 메타스토어 라 불리는 데이터베이스에 저장

  • 하이브를 처음 사용시, 사용자의 로컬 머신에 메타스토어를 설치하는 것이 좋음(다른 사용자와 명세 공유 불가)

  • 실제 운영 환경에서는 공유 원격 메타스토어 방식

Hive와 하둡 버전의 호환성

하이브의 모든 배포판은 여러 버전의 하둡과 문제없이 작동하도록 설계

  • 하이브에 사용할 하둡의 버전을 알려주지 않아도 사용 가능

  • 하둡의 실행 파일이 실행 경로(PATH)에 있고 HADOOP_HOME 환경변수가 설정되어 있으면 가능

Hive 설치 순서

  1. Java Install
$ java -version
  1. Hadoop Install
# 환경변수 설정
$ vi .bash_profile
.bashrc
# HIVE_HOME 환경 변수 등록
export HADOOP_HOME='~/apache-hadoop-<x.y.z>-bin' 

# PATH 환경 변수 등록
export PATH=$PATH:$HIVE_HOME/bin

# 하둡 Li 경로
export CLASSPATH=$CLASSPATH:/usr/local/hadoop/lib/* 
$ hadoop
$ start-yarn.sh
$ start-dfs.sh

Hadoop Browser: http://[hadoop host]:50070/

Hadoop Cluster: http://[hadoop host]:8088/

  1. Hive Download

  • Apach Hive 검색 및 다운 https://hive.apache.org/general/downloads/

  • Hadoop 분산 프로그래밍 모델에 따라 버전이 다르다

  • Hadoop 2.0버전부터는 MapReduce 분산처리 모델이 Deprecated. Tez같은 다른 분산 프로그래밍 모델 지원

  • apach-hive-[버전]-bin.tar.gz 다운로드

  1. Hive 설치
# 압축 해제
$ tar -zxvf apach-hive-[버전]-bin.tar.gz

# 환경변수 설정
$ vi .bash_profile
.bashrc
# HIVE_HOME 환경 변수 등록
export HIVE_HOME=~/apache-hive-<x.y.z>-bin

# PATH 환경 변수 등록
export PATH=$PATH:$HIVE_HOME/bin

# 하이브 Lib 경로
export CLASSPATH=$CLASSPATH:/usr/local/hive/lib*
# 환경설정 적용
$ source .bash_profile
  1. Hive 환경설정 (hive-env.sh)

  • conf는 하이브 환경설정 파일들을 모아놓은 폴더

  • template으로 시작하는 파일들은 처음 시작시에 환경설정을 쉽게 잡아주기 위한 역할

  • **복사 후 수정할 곳만 수정해서 사용

$ cd $HIVE_HOME/conf

$ cp hive-env.sh.template hive-env.sh

$ vi hive-env.sh

$ export HADOOP_HOME = /usr/local/hadoop
  1. 데이터베이스 설치
  • 하이브에서 지원하는 데이터베이스 : Oracle, MySQL, MariaDB, Derby..
  1. Hive 메타스토어 환경 설정
  • ojdbc6.jar 복사, $Hive_HOME/lib 경로로 복사
  • DB 드라이버를 하이브 라이브러리 폴더에 복사
  • 기본 환경설정 파일들을 이용해서 hive-site.xml 생성
# Hive config 폴더 이동 
$ cd $HIVE_HOME/config

# hive-default.xml.template 복사
$ cp hive-default.xml.template hive-site.xml

# hive-site.xml 편집
$ vi hive-site.xml
  • hive-site.xml 편집
hive-site.xml
<property>
	<name>javax.jdo.option.ConnectionURL</name>
	<value>jdbc:oracle:thin:@//{hive server ip}/xe</value>
</property>
<property>
	<name>javax.do.option.ConnectionDriveName</name>
	<value>oracle.jdbc.driver.OracleDrive</value>
</property>
