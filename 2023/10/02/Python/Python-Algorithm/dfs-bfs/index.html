Publish Date:   2023-10-02
Algorithm

DFS (Depth First Search)

깊이 우선 탐색. 큐를 사용해서 구현한다.

인접행렬로 구현

재귀함수를 이용한 구현
adj = [ [0]*N for _ in range(N) ]

adj[0][1] = adj[0][7] = 1
adj[1][2] = adj[1][5] = 1
adj[2][3] = adj[2][4] = 1
adj[5][6] = 1
...

# 재귀함수
def dfs(root_node):

    for next_node in range(N):
            if adj[root_node][next_node] == 1:
            dfs(next_node)

dfs(0)

DFS & BFS

공통점

  1. 그래프 탐색 알고리즘

  2. 완전 탐색 알고리즘 (모든 경우의 수를 탐색) : 느리다

  3. 반드시 답을 찾을 수 있다.

차이점

  1. 탐색 순서가 다르다 (BFS : 최단 거리 탐색에 유리, 목표 노드를 최초로 만난 시점이 최단 거리임을 보장하고 더 이상 서칭을 안함)

  2. BFS는 최단거리를 보장한다. 중간에 가치치기를 할 수 있기 때문에 목표 노드를 조기 발견 시 중단 할 수 있다.

인접행렬 vs 인접리스트

인접행렬: O(V^2)
인접리스트: O(V+E) = O(MAX(V, E))

비교

간선이 적으면 적을수록 인접리스트가 메모리 측면에서 유리하다.
간선이 많으면 많을수록 인접행렬이 속도 측면에서 유리하다.

시간복잡도

인접행렬: O(V^2)
인접리스트: O(V + E)
