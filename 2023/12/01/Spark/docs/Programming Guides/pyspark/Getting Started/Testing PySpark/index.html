Testing PySpark

이 안내서는 PySpark 코드에 견고한 테스트를 작성하는 데 도움이 되는 참조 자료입니다.

PySpark 테스트 유틸리티 문서를 보려면 여기를 참조하세요. PySpark 내장 테스트 유틸리티 코드를 확인하려면 Spark 저장소를 여기에서 확인하세요. PySpark 테스트 프레임워크의 JIRA 보드 티켓을 보려면 여기를 참조하세요.

Build a PySpark Application

다음은 PySpark 애플리케이션을 시작하는 예제입니다. 이미 테스트할 준비가 된 애플리케이션이 있다면 다음 섹션인 “PySpark 애플리케이션 테스트”로 건너뛰십시오.

먼저, Spark 세션을 시작하세요.

from pyspark.sql import SparkSession
from pyspark.sql.functions import col

# Create a SparkSession
spark = SparkSession.builder.appName("Testing PySpark Example").getOrCreate()

다음으로, DataFrame을 생성하세요.

sample_data = [{
    {"name": "John    D.", "age": 30},
  {"name": "Alice   G.", "age": 25},
  {"name": "Bob  T.", "age": 35},
  {"name": "Eve   A.", "age": 28}    
}]

df = spark.createDataFrame(sample_data)

이제 DataFrame에 변환 함수를 정의하고 적용합시다.

from pyspark.sql.functions import col, regexp_replace

# Remove additional spaces in name
def remove_extra_spaces(df, column_name):
    # Remove extra spaces from the specified column
    df_transformed = df.withColumn(column_name, regexp_replace(col(column_name), "\\s+", " "))

    return df_transformed

transformed_df = remove_extra_spaces(df, "name")

transformed_df.show()

Testing your PySpark Application

이제 PySpark 변환 함수를 테스트해 봅시다.

한 가지 옵션은 단순히 결과 DataFrame을 시각적으로 확인하는 것입니다. 그러나 이는 큰 DataFrame이나 입력 크기에 대해서는 비실용적일 수 있습니다.

더 나은 방법은 테스트를 작성하는 것입니다. 아래는 코드를 테스트하는 몇 가지 예제입니다. 아래 예제들은 Spark 3.5 이상 버전에 적용됩니다.

이 예제들은 전체적이지 않습니다. 왜냐하면 unittest나 pytest 대신 사용할 수 있는 다양한 다른 테스트 프레임워크 대안이 있기 때문입니다. 내장 PySpark 테스트 유틸리티 함수는 독립적으로 작동하기 때문에 어떤 테스트 프레임워크나 CI 테스트 파이프라인과도 호환될 수 있습니다.

Option 1: Using Only PySpark Built-in Test Utility Functions

간단한 임시 유효성 검사 케이스의 경우, assertDataFrameEqual 및 assertSchemaEqual과 같은 PySpark 테스트 유틸리티를 독립적인 컨텍스트에서 사용할 수 있습니다. 노트북 세션에서 PySpark 코드를 쉽게 테스트할 수 있습니다. 예를 들어, 두 개의 DataFrame 간의 동등성을 확인하려면:

import pyspark.testing
from pyspark.testing.utils import assertDataFrameEqual

# Example 1
df1 = spark.createDataFrame(data=[("1", 1000), ("2", 3000)], schema=["id", "amount"])
df2 = spark.createDataFrame(data=[("1", 1000), ("2", 3000)], schema=["id", "amount"])
assertDataFrameEqual(df1, df2)  # pass, DataFrames are identical
# Example2
df1 = spark.createDataFrame(data=[("1", 0.1), ("2", 3.23)], schema=["id", "amount"])
df2 = spark.createDataFrame(data=[("1", 0.109), ("2", 3.23)], schema=["id", "amount"])
assertDataFrameEqual(df1, df2, rtol=1e-1) # pass, DataFrames are approx equal by rtol

"""
rtolfloat, optional
  실제 값과 예상 값 사이의 부동 소수점 값에 대한 근사 동등성을 확인하는 데 사용되는 상대 허용 오차입니다. 기본값은 1e-5입니다.
"""

또한 두 개의 DataFrame 스키마를 간단히 비교할 수도 있습니다.

from pyspark.testing.utils import assertSchemaEqual
from pysaprk.sql.types import StructType, StructField, ArrayType, DoubleType

s1 = StructType([StructField("names", ArrayType(DoubleType(), True), True)])
s2 = StructType([StructField("names", ArrayType(DoubleType(), True), True)])

assertSchemaEqual(s1, s2)  # pass, schemas are identical

Option 2: Using Unit Test

더 복잡한 테스트 시나리오의 경우, 테스트 프레임워크를 사용하는 것이 좋습니다.

가장 인기 있는 테스트 프레임워크 옵션 중 하나는 단위 테스트입니다. 내장된 Python unittest 라이브러리를 사용하여 PySpark 테스트를 작성하는 방법을 알아보겠습니다. unittest 라이브러리에 대한 자세한 정보는 여기를 참조하세요: https://docs.python.org/3/library/unittest.html.

먼저 Spark 세션이 필요합니다. unittest 패키지에서 제공하는 @classmethod 데코레이터를 사용하여 Spark 세션을 설정하고 해제할 수 있습니다.

import unittest

class PySparkTestCase(unittest.TestCase):
    @classmethod
    def setUpClass(cls):
        cls.spark = SparkSession.builder.appName("Testing PySpark Example.").getOrCreate()

    @classmethod
    def tearDownClass(cls):
        cls.spark.stop()

이제 unittest 클래스를 작성해 봅시다.

from pyspark.testing.utils import assertDataFrameEqual

class TestTransformation(PySparkTestCase):
    def test_single_space(self):
        sample_data = [{"name": "John    D.", "age": 30},
                       {"name": "Alice   G.", "age": 25},
                       {"name": "Bob  T.", "age": 35},
                       {"name": "Eve   A.", "age": 28}]
        
        # Create a Spark DataFrame
        original_df = spark.createDataFrame(sample_data)

        # Apply the transformation function from before
        transformed_df = remove_extra_spaces(original_df, "name")

        expected_data = [{"name": "John D.", "age": 30},
        {"name": "Alice G.", "age": 25},
        {"name": "Bob T.", "age": 35},
        {"name": "Eve A.", "age": 28}]

        expected_df = spark.createDataFrame(expected_data)

        assertDataFrameEqual(transformed_df, expected_df)

실행되면 unittest는 이름이 “test”로 시작하는 모든 함수를 선택합니다.

Option 3: Using Pytest

우리는 또한 가장 인기 있는 Python 테스트 프레임워크 중 하나인 pytest로 테스트를 작성할 수도 있습니다. pytest에 대한 자세한 정보는 여기 문서를 참조하세요: https://docs.pytest.org/en/7.1.x/contents.html.

pytest fixture를 사용하면 테스트가 완료될 때 Spark 세션을 공유하고 해체할 수 있습니다.

import pytest

@pytest.fixture
def spark_fixture():
    spark = SparkSession.builder.appName("Testing PySpark Example").getOrCreate()
    yield spark

그런 다음 테스트를 다음과 같이 정의할 수 있습니다:

```python
import pytest
from pyspark.testing.utils import assertDataFrameEqual

def test_single_space(spark_fixture):
    sample_data = [{"name": "John    D.", "age": 30},
                   {"name": "Alice   G.", "age": 25},
                   {"name": "Bob  T.", "age": 35},
                   {"name": "Eve   A.", "age": 28}]

    # Create a Spark DataFrame
    original_df = spark.createDataFrame(sample_data)

    # Apply the transformation function from before
    transformed_df = remove_extra_spaces(original_df, "name")

    expected_data = [{"name": "John D.", "age": 30},
    {"name": "Alice G.", "age": 25},
    {"name": "Bob T.", "age": 35},
    {"name": "Eve A.", "age": 28}]

    expected_df = spark.createDataFrame(expected_data)

    assertDataFrameEqual(transformed_df, expected_df)

pytest 명령어로 테스트 파일을 실행하면 이름이 “test”로 시작하는 모든 함수를 선택합니다.

Putting It All Together!

함께 모든 단계를 살펴보겠습니다. Unit Test 예제를 통해요.

import unittest

from pyspark.sql import SparkSession
from pyspark.sql.functions import col, regexp_replace
from pyspark.testing.utils import assertDataFrameEqual

# Create a SparkSession
spark = SparkSession.builder.appName("Sample PySpark ETL").getOrCreate()

sample_data = [{"name": "John    D.", "age": 30},
  {"name": "Alice   G.", "age": 25},
  {"name": "Bob  T.", "age": 35},
  {"name": "Eve   A.", "age": 28}]

df = spark.createDataFrame(sample_data)

# Define DataFrame transformation function
def remove_extra_spaces(df, column_name):
    # Remove extra spaces from the specified column using regexp_replace
    df_transformed = df.withColumn(column_name, regexp_replace(col(column_name), "\\s+", " "))

    return df_transformed
pkg/test_etl.py
import unittest

from pyspark.sql import SparkSession

# Define unit test base class
class PySparkTestCase(unittest.TestCase):
    @classmethod
    def setUpClass(cls):
        cls.spark = SparkSession.builder.appName("Sample PySpark ETL").getOrCreate()

    @classmethod
    def tearDownClass(cls):
        cls.spark.stop()

# Define unit test
class TestTranformation(PySparkTestCase):
    def test_single_space(self):
        sample_data = [{"name": "John    D.", "age": 30},
                        {"name": "Alice   G.", "age": 25},
                        {"name": "Bob  T.", "age": 35},
                        {"name": "Eve   A.", "age": 28}]

        # Create a Spark DataFrame
        original_df = spark.createDataFrame(sample_data)

        # Apply the transformation function from before
        transformed_df = remove_extra_spaces(original_df, "name")

        expected_data = [{"name": "John D.", "age": 30},
        {"name": "Alice G.", "age": 25},
        {"name": "Bob T.", "age": 35},
        {"name": "Eve A.", "age": 28}]

        expected_df = spark.createDataFrame(expected_data)

        assertDataFrameEqual(transformed_df, expected_df)
unittest.main(argv=[''], verbosity=0, exit=False)
